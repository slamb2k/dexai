# Heartbeat Checks

Periodic checks that run in the background to keep the agent aware.

## Morning Briefing
- Check calendar for today's events
- Summarize unread important emails
- Review pending tasks from yesterday

## Monitoring
- Check if backup completed successfully
- Verify API endpoints are responding
- Check for security alerts

## Project Status
- Review any failing CI/CD builds
- Check for new issues or pull requests
- Monitor resource usage trends
